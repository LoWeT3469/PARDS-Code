# PARDS Code Repository

This repository keeps the existing code tree structure and tracks:

- `02-DT-CWPT/`: shared DTCWPT MATLAB utilities
- `2-PARDS/`: PARDS V1
- `3-PARDS_V2/`: PARDS V2
- `4-PARDS_V3/`: PARDS V3

Why one repo instead of three:

- the three PARDS versions are one project lineage
- `02-DT-CWPT` is shared across versions
- one repo keeps history, branching, and pushing simpler
- no need to duplicate or split shared utilities

The following are intentionally not tracked here: large data, generated outputs, Slurm logs, third-party repos, and unrelated code trees.
